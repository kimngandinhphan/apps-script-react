# Google Apps Script + React with React Router

# Quick start
Install the dependencies

```sh
npm install
```
Log in to clasp

```sh
clasp login
```
Set up

```sh
npm run setup
```

# Local Development
Start the project locally

```sh
npm run start:local
```

# Deploy
Build and deploy the project

```sh
npm run deploy
```
Start the project

```sh
npm run start
```
